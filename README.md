# test-project-1
![image](https://github.com/user-attachments/assets/bfe91ac6-096f-4598-880f-6ca813afcd00)

В проекте реализована демонстрация сбора ресурсов ботами.
Рандомные ресурсы спавнятся в рандомных местах пространства.
Есть две базы. У каждой базы свои боты, которые автоматически собирают ресурсы по заданному алгоритму.
Навигация ботов осуществляется посредством NavMesh агнетов.

Есть элементы UI позволяющие:
- изменять колличество ботов, которые могут быть у каждой базы
- изменять скорость передвижения ботов
- изменять скорость спавна ресурсов
- включать/выключать отрисовку пути ботов
- отображать ресурсы баз

Проект реализован с использованием архитектуры MVC.
Слой моделей отвечает за хранение данных игры и взаимодействие с ними.
Слой контроллеров отвечает за логику игры, обработку событий и отображение данных.
Слой представления отвечает за видимые объекты в сцене.

Все сущности требующие выполнение логики в Update() реализуют интерфейс ITickable и управляются централизованно через TickManager.
